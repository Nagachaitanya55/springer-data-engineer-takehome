# Springer Data Engineer Take-Home Assignment

This project implements a data processing pipeline for a referral reward system.
The pipeline ingests multiple datasets related to users, referrals, transactions, and rewards, performs data profiling and transformation, and generates a final referral reward report.

The goal of this project is to simulate a real-world **data engineering workflow**, including:

* Data ingestion
* Data profiling and validation
* Data transformation
* Business rule implementation
* Final analytical report generation

Dataset Description

The project uses multiple datasets representing different parts of a referral system.

| File                         | Description                                  |
| ---------------------------- | -------------------------------------------- |
|  lead_log.csv                | Logs of leads generated by users             |
|  paid_transactions.csv       | Records of successful transactions           |
|  referral_rewards.csv        | Reward configuration and rules               |
|  user_logs.csv               | User activity logs                           |
|  user_referrals.csv          | Mapping between referrers and referred users |
|  user_referral_logs.csv      | Detailed logs of referral events             |
|  user_referral_statuses.csv  | Status updates for referral processes        |

Additional metadata is available in:
docs/data_dictionary.csv

Scripts

1. Data Profiling Script

scripts/data_profiling.py

This script performs exploratory data profiling on the datasets including:

* Column statistics
* Missing value detection
* Data type inspection
* Basic validation checks

Output:
output/data_profiling_report.csv


2. Referral Pipeline Script
scripts/main_referral_pipeline.py

This is the main data processing pipeline that:

1. Loads all datasets
2. Cleans and validates data
3. Joins referral and transaction datasets
4. Applies referral reward logic
5. Generates the final referral reward report

Output:
output/final_referral_report.csv

Installation

Clone the repository:

bash
git clone <repository_url>
cd Springer_takehome

Install required dependencies:

bash
pip install -r requirements.txt

Running the Project

Step 1 — Run Data Profiling

bash
python scripts/data_profiling.py


This will generate:

output/data_profiling_report.csv

Step 2 — Run the Referral Processing Pipeline

bash
python scripts/main_referral_pipeline.py

This will generate:
output/final_referral_report.csv
